# Chocolate Sales Analysis Dashboard

## Project Overview

This project analyzes over **85,000 chocolate sales records** using **MySQL** and **Power BI** to uncover business insights through interactive dashboards.

## Tools Used

- SQL
- MySQL
- Power BI
- DAX
- Power Query
- Microsoft Excel

## Dashboard Features

- Total Sales KPI
- Total Orders
- Total Boxes Shipped
- Average Order Value
- Average Discount
- Marketing Spend
- Sales by Country
- Top Selling Products
- Sales by Salesperson
- Sales by Channel
- Monthly Sales Trend
- Interactive Filters & Slicers

##  Key Insights

- Brazil generated the highest revenue.
- Retail was the highest-performing sales channel.
- Mixed Assortment Box was the top-selling product.
- Sales increased significantly towards the end of the year.
